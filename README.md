## 🔧 Customization

### Adding New Links
1. Edit the appropriate JSON file in the `data/` directory
2. Follow the existing format:
```json
{
  "name": "Site Name",
  "url": "https://example.com",
  "warning": "⚠ WARNING: ... (optional)",
  "description": "Site description (optional)",
  "popularity": 85 (optional)
}
```

### Changing Colors
Edit the CSS custom properties in `styles/main.css`:
```css
:root {
  --primary: #1a1a2e;
  --secondary: #16213e;
  --accent: #0f3460;
  --text: #e94560;
  --highlight: #533483;
}
```

### Adding New Categories
1. Add category to `utils/data.js`
2. Create corresponding JSON data file
3. Update category icons and colors
