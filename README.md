### 💻 **Option 3: Local Development**
```bash
# Clone the repository
git clone https://gitlab.com/alessthegreatestgamer/ultimatelinks.git
cd ULTIMATELINKS

# Using Python (recommended)
python -m http.server 8080

# Using Node.js
npx serve . -p 8080

# Using Live Server
npx live-server --port=8080
```

Simply open `index.html` in your web browser

## 🔧 **Customization Guide**

### ➕ **Adding New Links**
Edit the `loadCategories()` function in `script.js`:

```javascript
{
    name: 'CATEGORY NAME',
    icon: '🎯',
    count: 1,
    description: 'Category description',
    links: [
        {
            name: 'Site Name',
            url: 'https://example.com',
            description: 'Site description'
        }
    ]
}
```

### 🎨 **Theme Customization**
Edit CSS variables in `styles.css`:

```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #e94560;
    --bg-gradient: linear-gradient(135deg, #1a1a2e 0%, #16213e 50%, #0f3460 100%);
    --card-bg: rgba(255, 255, 255, 0.05);
    --text-primary: #ffffff;
    --text-secondary: #94a3b8;
}
```

### ⚙️ **Settings Configuration**
The settings panel allows customization of:
- **🎨 Theme selection** (multiple color schemes)
- **⚡ Animation preferences**
- **📱 Display options**
- **🔔 Notification settings**
- **⚡ Performance optimization**
- **♿ Accessibility features**

### 🎪 **Welcome Popup Customization**
Edit `simple_popup.js` to modify welcome experience:

```javascript
// Update popup content
popupContent.innerHTML = `
    <h2>Your Custom Title</h2>
    <p>Your custom message</p>
    <div class="stats">
        <span class="stat">70+ Links</span>
        <span class="stat">10 Categories</span>
    </div>
`;
```
